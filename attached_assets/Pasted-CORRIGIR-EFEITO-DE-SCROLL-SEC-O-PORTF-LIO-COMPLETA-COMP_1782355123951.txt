CORRIGIR EFEITO DE SCROLL — SECÇÃO PORTFÓLIO COMPLETA
COMPORTAMENTO GERAL CORRECTO
O efeito tem 2 fases distintas:
FASE 1 — Entrada da secção:

O utilizador está a fazer scroll normal na secção anterior (A NOSSA METODOLOGIA)
Quando chega ao fim da secção anterior, essa secção fica completamente parada/fixa
A secção de Portfólio começa a subir por baixo da secção anterior, tapando-a progressivamente
Quando o Portfólio ocupa 100% do ecrã, a secção anterior desaparece completamente

FASE 2 — Dentro da secção Portfólio:

O Projecto [001] está visível e fica fixo/imóvel
O utilizador continua a fazer scroll → Projecto [002] sobe por baixo e tapa o [001]
O [001] fica completamente parado enquanto o [002] sobe
Quando [002] ocupa 100% → fica fixo → [003] sobe por baixo
E assim sucessivamente

javascript// Controlar posição de cada slide com base no scroll
window.addEventListener('scroll', () => {
  const wrapper = document.querySelector('.portfolio-wrapper');
  const wrapperTop = wrapper.getBoundingClientRect().top;
  const wrapperHeight = wrapper.offsetHeight;
  const vh = window.innerHeight;

  // Progresso total dentro do wrapper (0 a 1)
  const totalProgress = Math.max(0, Math.min(1,
    -wrapperTop / (wrapperHeight - vh)
  ));

  const slides = document.querySelectorAll('.project-slide');
  const numSlides = slides.length;

  slides.forEach((slide, index) => {
    if (index === 0) {
      // Primeiro slide fica fixo desde o início
      slide.style.transform = 'translateY(0)';
      return;
    }

    // Cada slide subsequente tem o seu próprio progresso
    const slideProgress = Math.max(0, Math.min(1,
      (totalProgress * numSlides) - index
    ));

    // Move de 100% (baixo) para 0% (posição final)
    const yPos = (1 - slideProgress) * 100;
    slide.style.transform = `translateY(${yPos}%)`;
  });
});
