EFEITO HOVER — LUZ POR CÉLULA
Quando o mouse passa sobre qualquer célula da grid:

Aplicar um glow/radial gradient amarelo-dourado centrado no cursor dentro da célula:
cssbackground: radial-gradient(circle at center, rgba(245, 166, 35, 0.25) 0%, transparent 70%);

Adicionar uma borda amarela fina (1px solid #F5A623) na célula em hover
Transição suave: transition: all 0.3s ease
O efeito deve seguir o cursor dentro da célula usando JavaScript (mousemove event), atualizando a posição do radial-gradient dinamicamente:
javascriptcell.addEventListener('mousemove', (e) => {
  const rect = cell.getBoundingClientRect();
  const x = ((e.clientX - rect.left) / rect.width) * 100;
  const y = ((e.clientY - rect.top) / rect.height) * 100;
  cell.style.background = `radial-gradient(circle at ${x}% ${y}%, rgba(245,166,35,0.25) 0%, transparent 70%)`;
  cell.style.border = '1px solid #F5A623';
});
cell.addEventListener('mouseleave', () => {
  cell.style.background = 'transparent';
  cell.style.border = '';
});
